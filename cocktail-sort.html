原理： 雞尾酒排序法是雙向的 Bubble Sort。先由左到右把大值推到右端，再由右到左把小值推到左端。

⏱ 時間複雜度：最佳 O(n)，平均 / 最差 O(n²)，空間複雜度 O(1)。

  let start = 0, end = arr.length - 1;
  let swapped = true;
  while (swapped) {
    swapped = false; // 左到右
    // 右到左再做一次
  }
}
