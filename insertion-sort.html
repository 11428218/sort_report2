原理： 插入排序法會逐一將元素插入到已排序區域的正確位置。 每次插入後，已排序區域擴大一格，直到所有元素排序完成。

⏱ 時間複雜度：最差 / 平均為 O(n²)，最佳為 O(n)（已排序陣列）。

function insertionSort(arr) {
  for (let i = 1; i < arr.length; i++) {
    let key = arr[i];
    let j = i - 1;
    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = key;
  }
}
